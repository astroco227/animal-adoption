# Animal Adoption Website

A full-stack web application designed to manage animal adoptions, streamline administrative tracking, and connect pets with loving families.

## 🚀 Features
* **User Authentication:** Secure registration and login for users.
* **Pet Profiles:** Browse available animals with details.
* **Adoption Requests:** Users can submit forms to adopt specific animals.
* **Admin Dashboard:** Full CRUD operations to manage pet listings and requests.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Database:** MySQL (XAMPP environment)
